# ResearchMind-AI

## Overview
ResearchMind-AI is an AI-powered multi-agent research assistant developed using IBM watsonx.ai, IBM Granite Models, IBM Bob, and Retrieval-Augmented Generation (RAG). It helps researchers, students, and professionals analyze research papers, generate literature reviews, identify research gaps, and forecast emerging research trends.

## Features
- AI-powered literature review generation
- Research gap analysis
- Trend forecasting
- Intelligent research recommendations
- Retrieval-Augmented Generation (RAG)
- Interactive research assistant

## Technologies Used
- IBM Bob
- IBM watsonx.ai
- IBM Granite Models
- Flask (Python)
- RAG
- In-Memory Vector Store
